# AI Student Performance Predictor

## Overview

AI Student Performance Predictor is a Machine Learning web application built using Flask and Python.
The system predicts student final marks based on:

* Study Hours
* Attendance
* Previous Marks

The application uses Machine Learning algorithms to analyze performance and generate predictions with smart suggestions.

---

## Features

* Machine Learning Prediction
* Flask Web Application
* Responsive Bootstrap UI
* SQLite Database Integration
* Prediction History
* Smart Suggestions
* PDF Report Generation
* Multiple ML Algorithms Comparison

## Installation

### Clone Repository

```bash
git clone <repository_link>
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Train Model

```bash
python train_model.py
```

### Run Flask Application

```bash
python app.py
```
